### Build Setup

> for server directory

``` bash
# move directory 
cd server

# install dependencies
npm install

# start server at localhost:8080
npm run start
```


> for client directory

``` bash
# move directory 
cd client

# install dependencies
npm install

# serve with hot reload at localhost:8080 or 8081
npm run serve

# build for production with minification (tored in /server/dist)
npm run build
```
